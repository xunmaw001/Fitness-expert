# Fitness-expert
“健身达人”微信小程序，是你的贴身私教，随时陪练。它有海量专业健身课程，涵盖塑形燃脂、增肌训练、舒缓拉伸等类型，不管是健身小白，还是运动大神，都能找到适配课程。小程序还提供个性化运动计划，根据你的身体状况、目标量身定制。运动过程中，动作指导语音实时播报，规范你的动作。并且，它能自动记录训练数据，方便复盘。使用“健身达人”，让你轻松在家健身，塑造健康体魄 。
